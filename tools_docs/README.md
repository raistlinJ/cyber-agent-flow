# Tool Guides

These guides document complex or high-risk tools exposed to the agent.

The agent loop auto-loads only the guides for currently enabled tools.

## Current guides

- `msf_run.md`
- `RIPv2.md`
- `ospf_sniff.md`
- `shell.md`
- `shell_extended.md`
- `shell_sequence.md`
- `shell_dangerous.md`
- `interactive_sessions.md`

## Authoring notes

- Keep guidance operational, not theoretical.
- Prefer short examples that match real tool schema and constraints.
- Include defaults, guardrails, and common failure patterns.
- Use exact tool names as shown in `kali_tools.json` or built-in MCP tool names.
